# MYGURUV Project

MYGURUV is a web-based application designed to help students learn effectively.

## Features
- User-friendly interface
- Student-focused learning modules
- Simple and clean design

## Technologies Used
- Python (Flask)
- HTML
- CSS
- MySQL

## How to Run
1. Install Python
2. Run `app.py`
3. Open browser and visit `http://127.0.0.1:5000`
